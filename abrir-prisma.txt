1. Certifique-se de que o Docker esteja em execução e que o serviço da API esteja ativo. (docker ps)
2. caso contrário execute com docker compose up --build -d 
3. execute o comando: docker compose exec api npx prisma studio --port 5555 --browser none
4. acesse o endereço http://(ip ou dominio):5555 para visualizar o Prisma Studio e interagir com o banco de dados.
